# 🎬 memoForMovie

> 영화를 기록하고, 데이터를 통해 취향을 분석하며, 업적으로 보상받는 개인 영화 아카이브 웹앱

---

## 📌 프로젝트 소개

**memoForMovie**는 영화를
기록하고 분석하며 개인의 취향을 축적할 수 있도록 만든 영화 기록 웹 애플리케이션입니다.

이 프로젝트는 “기록을 남기는 행위” 자체에 집중했습니다..  
영화를 보고 끝나는 것이 아니라,

- 언제 봤는지
- 어떤 감정을 느꼈는지
- 어떤 취향을 가지고 있는지

를 데이터로 쌓아가는 경험을 만드는 것이 목표입니다.

---
## 🛠 기술 스택

- Vue 3
- TypeScript
